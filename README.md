# ✦ ReminderAndHomework


реф : https://github.com/Sanchess0-o/Eco-helper-bot (есть напоминания) . 

Телеграм бот для управления дз с ии помощником.

Бд - добавляешь задание с помощью команды, бот предлагает в какой предмет сохранить(если какого-то нету то можно добавить) и дедлайн, бот напоминает за один день до делайна.можно установить собственное

ии объясняет что тебе не понятно в дз.



Управление домашними заданиями
Команда /add — добавить новое задание
Бот спрашивает предмет -> показывает список существующих предмев или предлагает создать новый
Бот запрашивает описание задания и дедлайн
Задания хранятся в БД, привязаны к конкретному пользователю


Команда /list — список всех активных заданий с дедлайнами


Команда /done — отметить задание как выполненное


Команда /delete — удалить задание


Команда /subjects — просмотр и редактирование списка предметов

библиотека для напоминания apscheduler
